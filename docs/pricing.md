# 定价

Codex 的定价方案

## 概述

Codex 提供两种使用方式：

1. **ChatGPT 订阅** - 包含在 ChatGPT Plus/Pro 套餐中
2. **API 计费** - 按使用量付费

## ChatGPT 订阅

每个 ChatGPT 套餐都包含 Codex 使用权限：

| 套餐 | 价格 | Codex 使用 |
|------|------|-----------|
| ChatGPT Free | 免费 | 有限使用 |
| ChatGPT Plus | $20/月 | 标准使用 |
| ChatGPT Pro | $200/月 | 优先使用 |

### 使用限制

- **Free 用户**: 每日有限的使用次数
- **Plus 用户**: 标准配额，适合日常开发
- **Pro 用户**: 高配额，适合重度使用

## API 计费

使用 API 密钥登录时，按标准 API 费率计费：

| 模型 | 输入价格 | 输出价格 |
|------|---------|---------|
| Codex Latest | $X / 1K tokens | $X / 1K tokens |
| Codex Fast | $X / 1K tokens | $X / 1K tokens |

> 查看最新的 API 定价：[platform.openai.com/pricing](https://platform.openai.com/pricing)

## Codex Cloud

Codex Cloud 包含在 ChatGPT 订阅中：

- **Plus 用户**: 每月有限的云会话
- **Pro 用户**: 更多的云会话配额

## 企业版

企业定制定价，联系销售团队获取报价。

企业功能包括：
- 无限使用
- 专属支持
- 定制化部署
- 数据驻留选项

## 如何查看用量

### ChatGPT 订阅用户

在 ChatGPT 设置中查看使用情况。

### API 用户

在 OpenAI Platform 仪表板查看：

1. 访问 [platform.openai.com/usage](https://platform.openai.com/usage)
2. 查看每日和每月用量
3. 设置用量警报

## 节省成本技巧

1. **使用快速模式** - 适合简单任务
2. **合理设置审批** - 减少不必要的确认
3. **批量处理任务** - 减少上下文切换
4. **清理长对话** - 避免过长的上下文
