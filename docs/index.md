## 为什么学习 Codex？

Codex 是 OpenAI 推出的 AI 编程助手，它能够：

- 🚀 **提升开发效率** - 自动完成代码、生成测试、重构代码
- 🧠 **理解项目上下文** - 感知整个代码库，提供精准建议
- 🔒 **安全优先** - 沙箱执行、权限控制、企业级安全功能
- 🌐 **无处不在** - CLI、IDE、Web、移动端，随时随地编程

## 学习路径

```
入门 → 核心概念 → 配置使用 → 高级功能 → 最佳实践
```

### 📚 推荐学习顺序

1. **快速开始** - 安装配置，快速上手
2. **核心概念** - 理解 Codex 的工作原理
3. **配置指南** - 个性化你的 Codex
4. **使用案例** - 实际应用场景
5. **最佳实践** - 高效使用技巧
