# 集成

与第三方工具和服务的集成，增强 Codex 的协作能力。

## 目录

- [GitHub](/integrations/github) - 代码审查和 PR 工作流
- [Slack](/integrations/slack) - 团队通知和协作
- [Linear](/integrations/linear) - 任务管理和问题追踪

## 核心功能

### GitHub 集成

- 自动代码审查
- PR 摘要和反馈
- 提交历史分析

### Slack 集成

- 实时通知
- 团队协作文案
- 自动化提醒

### Linear 集成

- 任务自动创建
- 问题追踪
- 进度同步
