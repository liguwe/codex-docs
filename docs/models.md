# 模型

Codex 支持的 AI 模型

## 概述

Codex 使用 OpenAI 最新的 AI 模型，针对代码理解和生成进行了优化。

## 可用模型

### Codex Latest

**模型 ID**: `codex-latest`

- **描述**: 最新、最强大的 Codex 模型
- **适用场景**: 复杂任务、深度代码理解
- **速度**: 标准
- **成本**: 标准

### Codex Fast

**模型 ID**: `codex-fast`

- **描述**: 优化速度的轻量级模型
- **适用场景**: 简单任务、快速响应
- **速度**: 快速
- **成本**: 较低

### Codex Economy

**模型 ID**: `codex-economy`

- **描述**: 经济实惠的模型
- **适用场景**: 大批量任务、预算有限
- **速度**: 标准
- **成本**: 最低

## 选择模型

### 按使用方式

**CLI 配置**:
```json
{
  "model": "codex-latest"
}
```

**App 设置**:
在设置中选择首选模型。

### 按任务类型

| 任务类型 | 推荐模型 |
|---------|---------|
| 复杂重构 | Latest |
| 新功能开发 | Latest |
| 简单修改 | Fast |
| 代码审查 | Fast |
| 批量处理 | Economy |

## 切换模型

### 临时切换

在对话中指定：
```
使用 codex-fast 模型帮我检查这个函数
```

### 永久切换

修改配置文件：
```json
{
  "model": "codex-fast"
}
```

## 模型能力

### 代码理解

- 多语言支持（Python、JavaScript、Go、Rust 等）
- 框架理解（React、Django、Spring 等）
- 架构模式识别

### 代码生成

- 函数实现
- 测试用例生成
- 文档编写
- 重构建议

### 调试辅助

- 错误分析
- 日志解读
- 性能优化建议
