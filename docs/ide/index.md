# IDE 集成

在编辑器中使用 Codex

## 概述

Codex 提供 VS Code 和 JetBrains 系列 IDE 的扩展，让你可以在熟悉的编辑环境中使用 AI 能力。

## 支持的 IDE

| IDE | 支持状态 |
|-----|---------|
| Visual Studio Code | ✅ 完全支持 |
| IntelliJ IDEA | ✅ 完全支持 |
| PyCharm | ✅ 完全支持 |
| WebStorm | ✅ 完全支持 |
| GoLand | ✅ 完全支持 |
| RustRover | ✅ 完全支持 |
| PhpStorm | ✅ 完全支持 |

## 快速开始

### VS Code 安装

1. 打开扩展面板
2. 搜索 "Codex"
3. 点击安装
4. 登录账户

### JetBrains 安装

1. 打开设置 → 插件
2. 搜索 "Codex"
3. 点击安装
4. 重启 IDE

## 主要功能

### 1. 侧边栏对话

在 IDE 侧边栏与 Codex 对话：
- 代码问题
- 功能实现
- 代码审查

### 2. 行内建议

直接显示代码建议：
- 代码补全
- 问题修复
- 重构建议

### 3. 右键菜单

选中文本后右键：
- 解释代码
- 生成注释
- 重构代码
