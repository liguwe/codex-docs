# Codex App

Codex 桌面应用使用指南

## 概述

Codex App 是官方桌面应用，提供图形界面，适合日常开发使用。

## 支持平台

| 平台 | 版本 | 架构 |
|------|------|------|
| macOS | 12.0+ | Apple Silicon / Intel |
| Windows | 10+ | x64 / ARM64 |
| Linux | 主流发行版 | x64 (测试中) |

## 下载和安装

### macOS

```
1. 访问下载页面
2. 下载 macOS 版本
3. 拖拽到应用程序文件夹
4. 首次运行需要允许"未知开发者"
```

### Windows

```
1. 访问下载页面
2. 运行安装程序
3. 按提示完成安装
```

## 快速开始

### 1. 登录

- 使用 ChatGPT 账户登录
- 或使用 API 密钥登录

### 2. 选择项目

- 打开项目文件夹
- 或从最近项目选择

### 3. 开始对话

- 在聊天窗口输入问题
- Codex 会分析并回答

## 主要功能

- **智能对话** - 理解代码上下文
- **代码编辑** - 直接修改文件
- **文件浏览** - 查看项目结构
- **终端集成** - 运行命令
- **Git 集成** - 查看变更
