# App 设置

Codex App 的配置选项

## 概述

本页面介绍 Codex App 的各项设置选项，帮助你定制个性化的使用体验。

## 常规设置

### 外观

#### 主题

- **浅色** - 明亮主题
- **深色** - 黑暗主题
- **系统** - 跟随系统

#### 字体

```
字体大小：12px - 24px
字体家族：等宽字体选择
行高：1.2 - 2.0
```

### 语言

支持的语言：
- 简体中文
- 英语
- 日语

## 模型设置

### 默认模型

选择默认使用的模型：
- Codex Latest
- Codex Fast
- Codex Economy

### 模型参数

```
温度 (Temperature): 0.0 - 1.0
  - 低值：更确定
  - 高值：更创意

最大 Token: 1024 - 32768
  - 控制响应长度
```

## 项目设置

### 项目选择

```
1. 点击"打开项目"
2. 选择项目文件夹
3. 添加到最近项目
```

### 项目特定设置

每个项目可以有独立设置：
- 模型选择
- 规则配置
- 环境变量

## 编辑设置

### 自动保存

```
☐ 自动保存更改
  延迟：1s - 60s
```

### 格式化

```
☐ 保存时格式化
格式化器：Prettier / Black / gofmt
```

### 代码补全

```
☐ 启用 AI 补全
触发延迟：0ms - 1000ms
```

## 终端设置

### Shell 选择

```
macOS/Linux: bash / zsh / fish
Windows: PowerShell / CMD / WSL
```

### 终端外观

```
字体：等宽字体

颜色主题：选择配色
```

## Git 设置

### 自动提交

```
☐ Codex 修改前自动创建检查点
检查点前缀："checkpoint: "
```

### Git 用户

```
用户名：Your Name
邮箱：your@email.com
```

## 审批设置

### 审批模式

| 模式 | 说明 |
|------|------|
| 自动 | 所有操作自动批准 |
| 默认 | 根据风险级别决定 |
| 手动 | 所有操作需要确认 |

### 危险操作

配置哪些操作需要审批：
- 删除文件
- 执行命令
- 网络请求
- 写入敏感文件

### 记忆管理

### 记忆同步
