# Windows 版 App

Codex Windows 应用的特殊功能

## 概述

Codex Windows 应用针对 Windows 系统进行了优化，提供原生体验。

## 系统要求

### 最低要求

| 要求 | 说明 |
|------|------|
| 操作系统 | Windows 10 版本 1903+ |
| 架构 | x64 或 ARM64 |
| 内存 | 4 GB RAM |
| 存储 | 1 GB 可用空间 |

### 推荐配置

| 要求 | 说明 |
|------|------|
| 操作系统 | Windows 11 |
| 处理器 | 四核 2.4 GHz+ |
| 内存 | 8 GB RAM |
| 存储 | SSD |

## Windows 特有功能

### 1. Windows Terminal 集成

```
默认使用 Windows Terminal
支持 PowerShell / CMD / WSL
```

### 2. 文件资源管理器集成

```
右键菜单:
- "用 Codex 打开"
- "在此打开 Codex"
```

### 3. 任务栏功能

```
任务栏右键菜单:
- 最近项目
- 新建对话
- 退出
```

### 4. 文件关联

```
自动关联:
- .js, .ts, .py 等代码文件
- 右键"编辑"使用 Codex
```

## WSL 集成

### WSL 2 支持

```
1. 安装 WSL 2
2. 在 Codex 中选择 WSL
3. 选择发行版
```

### 跨平台访问

```
访问 WSL 文件:
\\wsl$\Ubuntu\home\user\project

访问 Windows 文件:
/mnt/c/Users/user/project
```

## Windows 快捷键

### 系统快捷键

| 快捷键 | 功能 |
|--------|------|
| Win+Shift+C | 打开 Codex |
| Alt+Enter | 全屏切换 |
| Alt+Tab | 应用切换 |

### 编辑器快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+Alt+↑ | 向上复制行 |
| Ctrl+Alt+↓ | 向下复制行 |
| Ctrl+D | 选择相同词 |

## 通知中心

### Windows 通知

```
设置 → 系统 → 通知
启用 Codex 通知
```

### 通知类型

- 任务完成
- 错误提醒
- 更新提醒

```

```

### 用户账户控制

```

☐ 自动下载安装
```

### 更新频道

```

- 稳定版
- 测试版
- 开发版
```

## 故障排除

### 常见问题

| 问题 | 解决方案 |
|------|---------|
| 启动慢 | 禁用启动扫描 |
| 权限错误 | 以管理员运行 |
| 中文乱码 | 设置 UTF-8 编码 |

### 日志位置

```
日志文件:
%APPDATA%\Codex\logs\main.log
```

### 重置应用

```
设置 → 应用 → Codex
→ 高级选项 → 重置
```
