# 故障排除

Codex App 常见问题和解决方案

## 安装问题

### 无法安装

**问题**: 安装程序无法运行

**解决方案**:
```
1. 右键 → 以管理员身份运行
2. 检查磁盘空间
3. 关闭杀毒软件重试
```

### 安装后无法启动

**问题**: 点击图标无反应

**解决方案**:
```
1. 检查任务管理器
2. 结束所有 Codex 进程
3. 重新启动
4. 查看日志文件
```

## 登录问题

### 无法登录

**问题**: 登录按钮无响应

**解决方案**:
```
1. 检查网络连接
2. 清除浏览器缓存
3. 尝试隐私模式
4. 检查防火墙设置
```

### 登录后掉线

**问题**: 频繁掉线

**解决方案**:
```
1. 检查网络稳定性
2. 更新到最新版本
3. 清除应用缓存
```

## 性能问题

### 应用卡顿

**问题**: 响应慢或卡死

**解决方案**:
```
1. 关闭不必要的项目
2. 清理缓存 (Cmd+Shift+Delete)
3. 减少打开的文件数量
4. 检查内存使用
```

### 高 CPU 使用

**问题**: CPU 占用过高

**解决方案**:
```
1. 查看活动监视器
2. 停止索引中的项目
3. 禁用自动保存
4. 重启应用
```

## 代码功能问题

### 代码不生成

**问题**: Codex 不生成代码

**解决方案**:
```
1. 检查 API 密钥
2. 检查网络连接
3. 查看配额使用
4. 尝试重新登录
```

### 代码执行失败

**问题**: 生成的代码执行失败

**解决方案**:
```
1. 检查运行环境
2. 查看错误日志
3. 重新生成
4. 手动修复
```

## 文件问题

### 无法打开文件

**问题**: 文件无法打开

**解决方案**:
```
1. 检查文件权限
2. 确认文件未被占用
3. 重启应用
```

### 文件保存失败

**问题**: 保存时出错

**解决方案**:
```
1. 检查磁盘空间
2. 检查文件权限
3. 关闭其他程序
4. 另存为新文件
```

## Git 问题

### Git 命令失败

**问题**: Git 操作失败

**解决方案**:
```
1. 检查 Git 是否安装
2. 检查 Git 配置
3. 在终端测试 Git
```

### 无法看到变更

**问题**: Git 面板不显示变更

**解决方案**:
```
1. 刷新 Git 面板
2. 检查 .gitignore
3. 重新打开项目
```

## 扩展问题

### 扩展不工作

**问题**: 安装的扩展无效

**解决方案**:
```
1. 禁用后重新启用
2. 卸载后重新安装
3. 检查扩展兼容性
```

### 扩展冲突

**问题**: 多个扩展冲突

**解决方案**:
```
1. 禁用所有扩展
2. 逐个启用排查
3. 更新到最新版本
```

## 日志位置

### 查找日志

```
macOS: ~/Library/Logs/Codex/
Windows: %APPDATA%\Codex\logs\
Linux: ~/.config/Codex/logs/
```

### 日志级别

```
设置 → 高级 → 日志级别:
- Error
- Warning
- Info
- Debug
```

## 重置和恢复

### 软重置

```
1. 关闭应用
2. 清除缓存
3. 重启应用
```

### 硬重置

```
1. 导出配置
2. 删除配置目录
3. 重新安装
4. 导入配置
```
