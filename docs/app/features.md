# 功能特性

Codex App 的功能特性介绍

## 概述

Codex App 提供丰富的功能，帮助你更高效地编程。

## 核心功能

### 1. 智能代码对话

与 Codex 讨论你的代码：

- 解释代码逻辑
- 询问架构设计
- 讨论最佳实践

### 2. 代码生成

让 Codex 帮你写代码：

- 生成新函数
- 创建组件
- 实现算法

### 3. 代码修改

直接编辑你的文件：

- 重构函数
- 修复 bug
- 添加功能

### 4. 文件管理

浏览和管理项目文件：

- 目录树浏览
- 快速搜索
- 文件预览

### 5. 终端集成

内置终端运行命令：

- 运行脚本
- 执行测试
- Git 操作

## 高级功能

### 6. Git 集成

版本控制功能：

- 查看变更
- 创建提交
- 推送代码

### 7. 多会话管理

同时处理多个任务：

- 并行对话
- 会话切换
- 历史记录

### 8. 上下文感知

理解项目结构：

- 自动索引
- 智能联想
- 跨文件引用

### 9. 自定义规则

定义 Codex 行为：

- 代码风格
- 命名规范
- 安全策略

## 效率工具

### 10. 快捷命令

| 快捷键 | 功能 |

| Cmd/Ctrl+K | 快速搜索 |
| Cmd/Ctrl+L | 新建对话 |
| Cmd/Ctrl+R | 重新生成 |

### 11. 代码片段

保存常用代码：

- 创建代码片段
- 快速插入
- 团队共享

### 12. 模板系统

使用模板加速开发：

- 组件模板
- 测试模板
- 文档模板
