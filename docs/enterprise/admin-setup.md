# 管理员设置指南

企业环境中的 Codex 管理员配置

## 概述

本指南帮助管理员在企业环境中部署和管理 Codex。

## 快速开始

### 1. 创建组织

1. 访问 OpenAI 企业门户
2. 创建新组织
3. 验证企业信息

### 2. 配置 SSO

设置单点登录：
- SAML 2.0
- OIDC
- Microsoft Entra ID

### 3. 邀请成员

添加用户到组织：
- 邮箱邀请
- 批量导入
- SCIM 自动同步

### 4. 设置策略

配置组织策略：
- 数据保留
- 访问控制
- 使用限制

## 策略管理

### 认证策略

```json
{
  "authentication": {
    "requireSSO": true,
    "requireMFA": true,
    "sessionTimeout": 3600
  }
}
```

### 使用策略

```json
{
  "usage": {
    "maxTokensPerDay": 1000000,
    "allowedModels": ["codex-latest"],
    "blockedFeatures": ["code-execution"]
  }
}
```

### 数据策略

```json
{
  "data": {
    "retentionDays": 90,
    "allowExport": false,
    "auditLogging": true
  }
}
```

## 故障排除

### 常见问题

| 问题 | 解决方案 |
|------|---------|
| 用户无法登录 | 检查 SSO 配置 |
| 权限不足 | 检查角色分配 |
| 策略不生效 | 检查策略优先级 |

## 最佳实践

### 1. 最小权限

```
只授予必要的权限
定期审查权限
```

### 2. 分层管理

```

```

### 3. 定期审计

```
每季度审计:
- 用户权限

- 使用日志
```
