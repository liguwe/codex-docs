# Codex Cloud

云端开发环境使用指南

## 概述

Codex Cloud 让你在浏览器中使用 Codex，无需本地安装，直接在云端执行任务。

## 访问方式

### 1. 在线访问

访问 [chatgpt.com/codex](https://chatgpt.com/codex) 开始使用。

### 2. GitHub 集成

在 GitHub PR 中标记 `@codex` 委托任务。

## 快速开始

### 步骤 1: 登录

使用 ChatGPT 账户登录。

### 步骤 2: 连接仓库

连接你的 GitHub 仓库。

### 步骤 3: 开始任务

输入任务描述，Codex 开始工作。

### 步骤 4: 审查结果

查看变更，创建 Pull Request。

## 主要功能

### 云端执行

- 无需本地环境
- 不消耗本地资源
- 24/7 可用

### GitHub 集成

- 直接创建 PR
- 自动关联 Issue
- 支持私有仓库

### 环境管理

- 预配置环境
- 自动安装依赖
- 隔离执行

## 使用场景

### 1. 快速原型
