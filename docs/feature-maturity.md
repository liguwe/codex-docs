# 功能成熟度

Codex 功能的成熟度等级和稳定性说明

## 概述

本页面跟踪 Codex 各项功能的成熟度状态，帮助用户了解功能的稳定性和推荐使用程度。

## 成熟度等级

### GA (General Availability) - 正式发布

**标准**:
- 功能完整且稳定
- 经过充分测试
- 有完整的文档
- 生产环境推荐使用

**标识**: 
```
✅ 正式发布
```

### Beta - 公开测试

**标准**:
- 功能基本完整
- 可能存在小问题
- 文档齐全
- 适合早期采用者

**标识**:
```
🟡 公开测试
```

### Alpha - 内部测试

**标准**:
- 功能开发中
- 可能有 Bug
- 文档可能不完整
- 不推荐生产使用

**标识**:
```
🔴 内部测试
```

### Deprecated - 已弃用

**标准**:
- 将被移除或替代
- 有迁移路径
- 设定移除日期

**标识**:
```
⚠️ 已弃用
```
